BUILD SETUP INSTRUCTIONS
========================

1. Increment "AssemblyVersion" and "AssemblyFileVersion" in "GlobalAssemblyInfo.cs"
2. Run "BuildSetup.bat"
  2.1. In MpeMaker, update:
    - Extension Informations
      - Version
      - Status
      - Release date
      - Tags
      - Extension description
      - This version description
  2.2. Build the setup
  2.3. Generate update xml
3. Upload the setup to Google Code (http://code.google.com/p/axis-camera-mpplugin/downloads/list)
4. Commit changes
5. Create tag (create it from the Log)
6. Merge branch changes to trunk
